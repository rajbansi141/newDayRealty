STEP 3: AUTHENTICATION SERVICE
===============================

File Created: frontend/src/services/authService.js
Location: d:\newDayRealty\frontend\src\services\authService.js

Purpose:
--------
Provide a clean interface for all authentication-related operations.
Handles API calls, token management, and user data persistence.

Methods Implemented:
--------------------

1. register(userData)
   - Registers a new user
   - Stores token and user data on success
   - Returns: { success, token, user } or { success: false, error }

2. login(email, password)
   - Authenticates user
   - Stores token and user data on success
   - Returns: { success, token, user } or { success: false, error }

3. logout()
   - Calls logout endpoint
   - Clears token and user data from localStorage
   - Always succeeds (clears local data even if API fails)

4. getCurrentUser()
   - Fetches current user data from backend
   - Requires valid token
   - Returns: { success, user } or { success: false, error }

5. updateDetails(userData)
   - Updates user profile information
   - Updates localStorage with new user data
   - Returns: { success, user } or { success: false, error }

6. updatePassword(currentPassword, newPassword)
   - Changes user password
   - Requires current password for verification
   - Returns: { success, message } or { success: false, error }

7. isAuthenticated()
   - Checks if user has a token
   - Returns: boolean

8. getStoredUser()
   - Gets user data from localStorage
   - Returns: user object or null

9. getStoredToken()
   - Gets token from localStorage
   - Returns: token string or null

Usage Examples:
---------------

import authService from '../services/authService';

// Register
const result = await authService.register({
  name: 'John Doe',
  email: 'john@example.com',
  password: 'password123',
  role: 'user'
});

if (result.success) {
  console.log('Registered:', result.user);
}

// Login
const loginResult = await authService.login('john@example.com', 'password123');

// Check authentication
if (authService.isAuthenticated()) {
  const user = authService.getStoredUser();
  console.log('Logged in as:', user.name);
}

// Logout
await authService.logout();

Benefits:
---------
1. Centralized authentication logic
2. Consistent error handling
3. Automatic token and user data management
4. Easy to use in components
5. Singleton pattern (one instance shared across app)
