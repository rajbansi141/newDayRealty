STEP 4: PROPERTY SERVICE
=========================

File Created: frontend/src/services/propertyService.js
Location: d:\newDayRealty\frontend\src\services\propertyService.js

Purpose:
--------
Handle all property-related API operations including CRUD, search, and filtering.

Methods Implemented:
--------------------

1. getProperties(filters = {})
   - Fetches all properties with optional filters
   - Filters: type, location, minPrice, maxPrice, bedrooms, bathrooms, etc.
   - Returns: { success, data: [properties], count, pagination }

2. getProperty(id)
   - Fetches single property by ID
   - Returns: { success, data: property }

3. getFeaturedProperties()
   - Fetches featured/highlighted properties
   - Returns: { success, data: [properties] }

4. searchProperties(searchTerm)
   - Full-text search across properties
   - Searches: title, description, location
   - Returns: { success, data: [properties] }

5. createProperty(propertyData)
   - Creates new property listing (requires authentication)
   - Returns: { success, data: property }

6. updateProperty(id, propertyData)
   - Updates existing property (requires authentication + ownership)
   - Returns: { success, data: property }

7. deleteProperty(id)
   - Deletes property (requires authentication + ownership)
   - Returns: { success, message }

8. approveProperty(id)
   - Approves pending property (admin only)
   - Returns: { success, data: property }

Usage Examples:
---------------

import propertyService from '../services/propertyService';

// Get all properties
const { data: properties } = await propertyService.getProperties();

// Get with filters
const filtered = await propertyService.getProperties({
  type: 'Villa',
  minPrice: 1000000,
  maxPrice: 5000000,
  bedrooms: 3,
  location: 'Kathmandu'
});

// Get single property
const { data: property } = await propertyService.getProperty('123');

// Search
const results = await propertyService.searchProperties('luxury villa');

// Create property
const newProperty = await propertyService.createProperty({
  title: 'Beautiful Villa',
  description: 'Amazing property...',
  price: 2500000,
  type: 'Villa',
  location: 'Kathmandu',
  bedrooms: 4,
  bathrooms: 3,
  area: 2800
});

// Update property
const updated = await propertyService.updateProperty('123', {
  price: 2600000
});

// Delete property
await propertyService.deleteProperty('123');

// Approve property (admin)
await propertyService.approveProperty('123');

Filter Options:
---------------
- type: 'Villa' | 'House' | 'Apartment' | 'Studio' | 'Commercial' | 'Land'
- status: 'For Sale' | 'For Rent' | 'Sold' | 'Rented'
- minPrice: number
- maxPrice: number
- bedrooms: number
- bathrooms: number
- location: string
- city: string
- featured: boolean
- approved: boolean (admin only)
- page: number (for pagination)
- limit: number (items per page)

Benefits:
---------
1. Clean separation of API logic from components
2. Consistent error handling
3. Easy to mock for testing
4. Type-safe with JSDoc comments (can add)
5. Reusable across multiple components
